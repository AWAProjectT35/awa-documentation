## Testing steps

**Step 1**

Call `GET /public/restaurants`. This should return:

```json
[
  {
    "restaurantId": 1111,
    "restaurantName": "Restaurant A",
    "managerName": "Marie Grünenfelder",
    "address": "Addresse 1",
    "opens": "8:00",
    "closes": "22:00",
    "image": "/url",
    "type": "Categorie A",
    "priceLevel": 3
  },
  {
    "restaurantId": 2222,
    "restaurantName": "Restaurant B",
    "managerName": "Marie Grünenfelder",
    "address": "Addresse 2",
    "opens": "10:00",
    "closes": "20:00",
    "image": "/url",
    "type": "Categorie B",
    "priceLevel": 1
  }
]
```

**Step 2**

Call `GET /public/restaurants` with the following queriy parameters:

- address: Addresse A
- restaurantName: Restaurant A
- price: 3
- type: Categoie A

this should return:

```json
[
  {
    "restaurantId": 1111,
    "restaurantName": "Restaurant A",
    "managerName": "Marie Grünenfelder",
    "address": "Addresse 1",
    "opens": "8:00",
    "closes": "22:00",
    "image": "/url",
    "type": "Categorie A",

]
```

- address: Addresse C
